# PetWorld
etWorld là website thương mại điện tử chuyên cung cấp thức ăn, phụ kiện, sản phẩm chăm sóc và dịch vụ dành cho thú cưng. Hệ thống được xây dựng với Laravel ở phía backend để quản lý dữ liệu, xử lý API, xác thực người dùng và quản trị sản phẩm. Phía frontend sử dụng React nhằm mang đến giao diện hiện đại, tốc độ tải nhanh và trải nghiệm mua sắm mượt
