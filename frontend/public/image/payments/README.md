# Logo phương thức thanh toán

Lưu 5 file ảnh logo vào đúng thư mục này với đúng tên dưới đây
(Footer đang trỏ tới các đường dẫn này):

| Logo        | Tên file cần lưu       | Đường dẫn dùng trong code        |
| ----------- | ---------------------- | -------------------------------- |
| Visa        | `visa.png`             | `/image/payments/visa.png`        |
| Mastercard  | `mastercard.png`       | `/image/payments/mastercard.png`  |
| PayPal      | `paypal.png`           | `/image/payments/paypal.png`      |
| Apple Pay   | `apple-pay.png`        | `/image/payments/apple-pay.png`   |
| Google Pay  | `google-pay.png`       | `/image/payments/google-pay.png`  |

Gợi ý: nên dùng ảnh PNG nền trong suốt, cao khoảng 30–40px.
Nếu muốn đổi tên file/đường dẫn, sửa mảng `PAYMENTS` trong
`src/components/layout/Footer.jsx`.
