# Ảnh bài viết (khối "Bài Viết Mới Nhất")

Lưu 5 file ảnh vào đúng thư mục này, đúng tên `.jpg` bên dưới
(BlogSeeder.php đang trỏ tới các đường dẫn này — mỗi bài một ảnh riêng):

| Bài viết                          | Tên file cần lưu                                  |
| --------------------------------- | ------------------------------------------------- |
| Chăm sóc chó con mới về nhà       | `cham-soc-cho-con-moi-ve-nha.jpg`                 |
| Chọn thức ăn hạt cho mèo          | `chon-thuc-an-hat-cho-meo-truong-thanh.jpg`       |
| Kinh nghiệm mua phụ kiện an toàn  | `kinh-nghiem-mua-phu-kien-an-toan.jpg`            |
| Lịch tắm & vệ sinh lông           | `lich-tam-va-ve-sinh-long-thu-cung.jpg`           |
| Kết hợp pate & thức ăn hạt        | `ket-hop-pate-va-thuc-an-hat.jpg`                 |

Lưu ý:
- Đường dẫn đầy đủ trong code: `/image/blogs/<tên-file>.jpg`
- Trang chủ chỉ hiển thị 3 bài mới nhất, nhưng nên đủ cả 5 file.
- Tỉ lệ ảnh đẹp nhất: ngang (khung hiển thị cao 200px, `object-fit: cover`).
- File `image1.png` hiện tại không khớp tên nào — xoá hoặc đổi tên theo bảng trên.
